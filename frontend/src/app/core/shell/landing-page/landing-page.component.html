Про проект

Ctor це веб-програма для будівельних компаній. Платформа дозволяє планувати та управляти будівництвом. Передбачається зберігання всієї проектної документації, інформації про матеріали та постачальників.

Ctor забезпечує розробку дорожніх карт та управління робочими процесами для архітекторів та менеджерів. Ctor показує поточний стан будівлі за допомогою інформаційних панелей та звітів. Ctor є багатокористувацькою системою B2B і забезпечує індивідуальну конфігурацію для кожного клієнта.
